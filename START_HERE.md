# С чего начать

## Если вы просто хотите посмотреть демо

Откройте опубликованную ссылку на сайт. Ничего устанавливать не нужно.

Если ссылка еще не опубликована, попросите автора проекта выложить демо на Vercel и отправить вам обычную ссылку вида:

```text
https://project-name.vercel.app
```

## Если вы запускаете проект у себя

1. Установите Node.js LTS: https://nodejs.org
2. Откройте PowerShell.
3. Выполните:

```powershell
cd C:\Projects\AI-упаковщик
npm install
npm run dev
```

4. Откройте в браузере:

```text
http://localhost:3000
```

## Если вы отправляете проект другим

Не отправляйте архив с кодом обычным пользователям. Им нужна готовая ссылка.

Опубликуйте проект на Vercel, затем отправьте ссылку на демо или сделайте QR-код на страницу `/qr`.

Для первых показов лучше включить код доступа на Vercel:

```text
DEMO_ACCESS_CODE=pilot-2026
```

После этого людям можно отправить:

```text
Ссылка: https://your-project-name.vercel.app
Код доступа: pilot-2026
```

Если код доступа не включать, сайт будет полностью открыт по ссылке.
